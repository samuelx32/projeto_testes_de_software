# projeto_testes_de_software

Projeto de estudos para colocar em prática conceitos da disciplina de Testes de Software.

Utilizando um sistema para auxílio ao cuidado familiar compartilhado da saúde de idosos,
Contendo Front end do usuário, Front end de gerenciamento do serviço, 
Servidor (Back end), API para comunicação entre front e back ends.
